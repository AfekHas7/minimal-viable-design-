# UI Rules for Otzmah Shketah (עוצמה שקטה)

Always follow these rules when making UI changes, creating new components, or designing new screens for this project.

1. **Read the Design System First**: Before designing any new screen or component, you MUST read `docs/design-system.md` and follow it closely.
2. **Language and Direction**: This is a Hebrew right-to-left (RTL) project. All new elements must support RTL properly and use Hebrew text.
3. **Aesthetic**: Maintain the premium, clean, and minimalistic tone from the landing page.
4. **Spacing and Hierarchy**: Use generous padding (especially `py-16 md:py-24` for sections) to allow elements to breathe.
5. **Color Palette**: Stick to the primary (`#0b3c5d`), accent (`#17b7ae`), and premium (`#d4a373`) colors defined in the system. Avoid arbitrary colors.
